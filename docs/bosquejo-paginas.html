📌 Recomendaciones de mejora (visual + arquitectura)

Visual
  • Definir layout maestro único para todos los módulos (header, breadcrumb y acciones).
  • Agregar semáforos de estado consistentes en ingreso, gestión e inventario.
  • Incluir panel de indicadores en dashboard con filtros por fecha/sala/cliente.
Arquitectura
  • Centralizar reglas de negocio en capa servicio para evitar lógica en controladores.
  • Incorporar auditoría transversal (usuario, fecha, origen) en todas las entidades críticas.
  • Preparar capa de consultas para dashboard (agregaciones) para mejorar rendimiento.
